# Claire's FoodieFanatix

A responsive food website built with **HTML**, **CSS**, **Bootstrap**, **JavaScript**, and **jQuery**.

## Overview

Claire's FoodieFanatix is a multi-page food showcase website created for a web development assessment.  
It presents featured meals, a food gallery, contact information, and a clean responsive layout.

A key feature of the site is the **sliding image carousel placed directly below the navigation bar** on the homepage.

## Features

- Responsive Bootstrap navigation bar
- **Sliding image bar / carousel below the navbar**
- Featured meal sections on the homepage
- Gallery page with food cards
- Dish modal popups for more details
- About page
- Contact page
- Dark mode toggle with `localStorage`
- Mobile-friendly layout

## Pages

- `index.html` - Homepage with hero carousel and featured meals
- `about.html` - Information about the website or brand
- `contact.html` - Contact details or contact form
- `gallery.html` - Food gallery with dish previews and modal details

## Technologies Used

- HTML5
- CSS3
- Bootstrap
- JavaScript
- jQuery
